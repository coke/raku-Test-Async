NAME
====



`Test::Async::Metamodel::ReporterHOW` - metaclass backing a reporter bundle

DESCRIPTION
===========



This class inherits from `Test::Async::Metamodel::BundleHOW` and adds implicit application of [`Test::Async::Reporter`](https://github.com/vrurg/raku-Test-Async/blob/v0.0.1/docs/md/Test/Async/Reporter.md) role.
